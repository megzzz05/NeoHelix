# Velora-AI 🌍

Velora-AI is an AI-powered language learning assistant developed by **NeoHelix**. The project uses Generative AI, LangChain, and LlamaIndex to analyze YouTube video transcripts and generate multilingual summaries, vocabulary lists, and AI-based MCQs to make language learning more interactive and efficient.

---

# Features

* 🎥 YouTube video transcript extraction
* 🌐 Multilingual summary generation
* 📚 Vocabulary extraction from transcripts
* 📝 AI-generated MCQs for learning
* 💡 Simple and interactive Streamlit interface

---

# Tech Stack

* Python
* Streamlit
* LangChain
* LlamaIndex
* Gemini API / OpenAI API
* YouTube Transcript API

---

# Installation

## Clone the repository

```bash
git clone https://github.com/megzzz05/NeoHelix
```

## Move into project folder

```bash
cd Velora-AI
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run the application

```bash
streamlit run app.py
```

---

# Project Workflow

1. User enters a YouTube video URL
2. Transcript is extracted using YouTube Transcript API
3. AI processes transcript using LLMs
4. Summary, vocabulary, and MCQs are generated
5. Results are displayed through Streamlit UI
